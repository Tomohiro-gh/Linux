## Homebrewを用いてフォントをインストールする
```sh
# serach
$ brew search (フォント名)
$ brew install --cask font-(フォント名)

## caskがオプションとして指定することになった点に注意

## ex Rictyの場合 ---------------
(base) ~ % brew search ricty
==> Formulae
redict

==> Casks
font-ricty-diminished

(base) ~ % brew install --cask font-ricty-diminished
## --------------------
```
